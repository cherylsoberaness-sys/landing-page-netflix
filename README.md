# Netflix Home landing page 

## Technical Decisions & Features

* **Layout Strategy:**
  The overall layout was built using **CSS Grid** to define the main page structure, while **Flexbox** was used      for internal components and content alignment.

* **Responsive Design:**  
  Basic responsiveness was implemented using **media queries**, focusing on mobile devices. Further improvements     will include additional breakpoints for a more complete responsive experience.

* **Carousels:**
  Two interactive carousels were implemented to display multiple movie images based on their category. Each          carousel includes animations to enhance the visibility and presentation of the posters.

* **Animations:**

  * Animated transitions were added to improve the interaction within the carousels.
  * A separate animation was applied to the main title to create a more dynamic visual impact.

* **Styling:**
  Custom styles were applied to buttons and UI elements, along with selected **typography** to maintain visual       consistency.

* **HTML Structure:**
  The project uses **semantic HTML** with a clear hierarchy to improve readability, accessibility, and               maintainability.

* **Methodology:**
  The codebase follows the **BEM (Block, Element, Modifier)** methodology to ensure scalable and organized CSS.

## 🚧 Future Improvements

* Refine the current use of **media queries**, as the implementation is currently focused on a single mobile resolution.

* Enhance UI elements by adding **icons to buttons** (e.g., play and info icons) to improve usability and visual clarity.

* Continue reducing reliance on hardcoded values to make the layout more flexible and scalable.
