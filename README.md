# Netflix Home landing page 

## 🧠 Technical Decisions & Features

* **Layout Strategy:**
  The layout was built using **CSS Grid** for the main structure and **Flexbox** for internal alignment.

* **Responsive Design:**
  Basic responsiveness was implemented using **media queries**, primarily targeting mobile devices, with plans to extend to additional breakpoints.

* **Carousels:**
  Two carousels were implemented to display movie categories, each with animations to enhance presentation.

* **Animations:**
  Animations were added to the carousels and main title to improve visual interaction.

* **Styling:**
  Custom button styles and typography were used to maintain visual consistency.

* **HTML Structure:**
  Semantic HTML was used to ensure a clear and accessible structure.

* **Methodology:**
  The project follows the **BEM** methodology for scalable and organized CSS.

---

## 📚 What I Learned

* Using **Flexbox for the main layout** led to reliance on hardcoded dimensions, making the layout harder to maintain.

* Switching to **CSS Grid** improved control and reduced the need for fixed values.

* I learned to use **Grid for layout** and **Flexbox for internal alignment**.

* I became more aware of **hardcoded values** and started thinking in terms of **layout as a system of boxes**.

---

## 🚧 Future Improvements

* Refine the current use of **media queries**, as the implementation is currently focused on a single mobile resolution.

* Enhance carousel interactivity using **JavaScript** for better navigation and user experience.

* Add **icons to buttons** (e.g., play and info) to improve usability.

* Continue reducing hardcoded values to make the layout more scalable.
